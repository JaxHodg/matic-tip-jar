## Development Commands

Deploy to Polygon MainNet:

`truffle migrate --network polygon`

Deploy to Polygon TestNet:

`truffle migrate --network mumbai`

Deploy to local emulator:

`truffle migrate --network development --reset`

Run Tests

`truffle test`
